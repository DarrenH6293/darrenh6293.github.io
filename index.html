Event Tracker Web App

A web application designed to help people stay on track of time-limited events across multiple places. The platform offers an interface to browse, follow, and post events, ensuring people never miss critical moments.

React TypeScript Next.js AWS Prisma PostgreSQL Cloudflare Vercel

Key Features:

  • No need to manually add reminders yourself
  • Personalized event recommendations
  • Filter events by categories and tags
  • Follow favorite threads and set reminders
  • Create own private thread to remind your own group

Technical Highlights:

  • Built with Next.js for server-side rendering
  • TypeScript implementation for type safety and code quality
  • Uses AWS Lambda for scalable API endpoints
  • Amazon RDS for reliable relational database and persistent data
  • Prisma ORM for type-safe database access and migrations
  • Cloudflare handles DNS resolution, enforces HTTPS, and provides DDoS protection and rate limiting
  • User authentication using Google OAuth
  • Vercel deploys and hosts the Next.js frontend
